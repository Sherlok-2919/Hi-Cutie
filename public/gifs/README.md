# GIF Files Setup

Please add the following GIF files to this folder:

1. **1.gif** - A waving/greeting animation (the cute waving character you showed)
2. **2.gif** - A blushing/shy animation (the cute blushing character you showed)

These GIFs will appear throughout the website as decorative floating elements.

## How to add them:
1. Save your GIF files
2. Rename them to `1.gif` and `2.gif`
3. Place them in this `public/gifs/` folder
4. The website will automatically load them

The GIFs appear in:
- Landing page (left and right sides)
- Cake section (floating around the cake)
- Message card (corner decorations)
- Countdown section (top and bottom)
- Photo gallery (floating in dark background)
